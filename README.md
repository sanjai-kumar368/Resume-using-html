## Resume Website Using HTML

## Overview
This repository contains the source code for a personal resume website built using HTML and CSS.  
The project showcases a clean and structured digital resume that can be viewed in a web browser.  
It is deployed using GitHub Pages and serves as a simple yet effective online presence for professional information.

Live Website:  
https://sanjai-kumar368.github.io/Resume-using-html/

---

## Purpose of the Project
The main goals of this project are:
- To create a web-based resume using core frontend technologies
- To learn and practice HTML layout and CSS styling
- To present personal information in an organized and visually accessible format
- To deploy a static website using GitHub Pages

---

## Features
- Clean and readable resume layout
- Sections including personal details, education, skills, and contact information
- Simple navigation through resume sections
- Static and lightweight
- Hosted on GitHub Pages
- Built without external frameworks

---

## Technologies Used
- HTML5 for structure and content
- CSS3 for styling and layout
- GitHub Pages for hosting and deployment

## Deployment
This resume website is hosted via GitHub Pages.

Deployment process:
- The repository contains an index.html file in the root
- GitHub Pages is enabled in repository settings
- The site is served from the main branch

Updates pushed to the repository automatically update the live site.
